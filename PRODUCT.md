# SideNook

SideNook is a floating terminal panel for macOS that lives at the edge of your screen. It snaps to any edge, collapses to a thin pill when you're not using it, and stays on top of every app — including full-screen windows.

It's built for people who run AI tools like Claude Code, Cursor, or shell-based agents alongside design or development work. The terminal is always one click away, and SideNook tells you when an agent is waiting on you — even when the panel is fully collapsed.

---

## Core features

**Always accessible, never intrusive**
SideNook floats above every app and space without stealing focus. Clicking into it won't interrupt what you're doing in Figma, Framer, Xcode, or the browser.

**Edge-snapping pill**
When collapsed, the panel becomes a thin indicator at the screen edge. It takes almost no space, snaps to any edge, and stays out of your way until you need it.

**Multiple terminal sessions**
Run as many shell sessions as you need in tabs. Each session tracks its current directory and title automatically.

**Named sessions with directory restore**
Sessions remember where you were. When you reopen SideNook, your tabs pick up from the same working directory — no hunting for the right folder.

**Attention detection**
When a running process — like Claude Code — pauses to ask you a question, the session dot turns yellow. You see it even when the panel is collapsed, so nothing blocks silently.

**Command reference panel**
A built-in, collapsible command cheat sheet lives in the sidebar. Common shell commands, clickable to send directly to the terminal.

**Quick-send snippets**
Saved commands in the sidebar that send to the active session in one click. Add your own: `make run`, `git status`, `claude`, `npm run dev`.

**Two layout modes**
Left sidebar or top bar. Switch based on how you work.

**Light and dark modes**
Full ANSI color support with palettes tuned to macOS Terminal.app. Both modes are first-class.

**Keyboard-first**
Every common action has a shortcut. `⌘T` new tab · `⌘W` close · `⌘[` / `⌘]` navigate · `⌘K` clear · `⌘+` / `⌘-` font size · and more.

---

## Who it's for

Designers and developers in AI-assisted workflows. If you're switching between a design tool and a terminal dozens of times a day — running builds, talking to agents, checking git state — SideNook reduces that friction without requiring you to give up screen space permanently.

---

## What makes it different

Most terminal apps are the main event. SideNook is a support character. It stays out of the way, surfaces when you need it, and signals when something's waiting on you. It's designed for people who use the terminal *alongside* other tools, not as their primary workspace.
